Что такое Power BI?
Power BI - это система аналитики, которая позволяет вам быстро оценить состояние вашего бизнеса для принятия управленческих решений.
На данный момент Power BI является развивающейся системой, функциональность которой постоянно расширяется.

Power BI Desktop

Создавайте яркие интерактивные отчеты с помощью средств визуальной аналитики, которые всегда под рукой.

Power BI Service

Поделитесь своими данными в облаке для совместной работы.

Power BI Mobile

Сохраняйте доступ к своим данным в любое время.
